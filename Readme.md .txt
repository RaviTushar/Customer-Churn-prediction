# Predicting Customer Churn in Subscription-Based Business

## Project Overview
This project predicts whether a subscription customer is likely to churn using customer profile, contract, billing, and service usage data.

## Problem Statement
Subscription businesses lose revenue when customers cancel. The goal is to build a machine learning model that identifies customers at high risk of churn so the business can take retention actions.

## Objectives
- Analyze customer churn patterns
- Clean and prepare the dataset
- Build ML classification models
- Evaluate model performance
- Recommend business actions

## Repository Structure
Explain Data, Source, Documentation, Reports, Logs, Models folders.

## How to Run
1. Install requirements
2. Add dataset to Data/raw/
3. Run cleaning script
4. Run model training script
5. Check results
